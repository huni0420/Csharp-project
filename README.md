# 포레스트(for-rest-orant)
##### 글 아래에 화면 스크린샷과 시연영상 링크가 있습니다

## :dart: Purpose
### 식당에서 주문관리와 매출 정산을 할수있는 프로그램이 있으면 좋을 것같다는 생각으로 만들게 되었습니다.
<br />

## :boom: main function
### ● 메뉴 선택시 가격과 수량을 입력하여 저장
### ● 테이블 별로 결제화면 구성
### ● 결제후 쌓인 데이터로 하루 매출을 출력
<br />
